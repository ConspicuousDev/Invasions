# Invasions
Spigot 1.8 plugin that adds dungeon-like Invasion events.

## Commands
- /invasions
  - Permission: invasions.commands.invasions
  - Main plugin command. 
- /invasionsconfig
  - Permission: invasions.commands.config
  - Config command.
